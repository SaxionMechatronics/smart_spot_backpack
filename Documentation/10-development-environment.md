## Development Environment

The development environment section describes the used hardware and software tools to manufatcture and assemble the backpack.

### Software Tools
* CURA (5.6.0)
* Solidworks 2023
* SOPHIA

### Hardware Tools
* 3D printer
* Rivet tool
* Soldering iron
* Crimp tool
* Wire stripper
* Allen key set
* Philips screwdriver
* Wrapping tools (Optional)
